## Welcome to the ShopLocal API

The ShopLocal API is organized around REST principles. It uses standard HTTP response codes, authentication, and verbs. Our API accepts JSON-encoded request bodies and returns JSON-encoded responses.

## Base URLs

Use the appropriate base URL depending on your environment:

- **Sandbox**: `https://sandbox-api.shoplocal.online/v1`
- **Production**: `https://api.shoplocal.online/v1`

## Authentication

All API requests require authentication using Bearer tokens. Include your API key in the Authorization header:

```bash
curl https://api.shoplocal.online/v1/businesses \
  -H "Authorization: Bearer YOUR_API_KEY"
```

Get your API keys from your [business dashboard](https://shoplocal.online/dashboard/settings/api).
